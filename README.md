# OpenCV Face Detection in Video

This project detects faces in a video using OpenCV and Haar Cascade Classifier.

## Features 

* Detect faces in video files
* Draw rectangles around detected faces
* Option to watch the processed video
* Counts detected faces
* Works with any video file path entered by the user

## Requirements

* Python 3.x
* OpenCV

## Installation

```bash
pip install opencv-python
```

## Run

```bash
python face_video_detection.py
```

Enter the video path when prompted.

## Technologies Used

* Python
* OpenCV
* Haar Cascade Classifier
